# Numerical Analysis Solver

A professional Python desktop application for solving Numerical Analysis problems, built as a university course project.

## Quick Start

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the application

```bash
python main.py
```

## Modules

| Module | Methods |
|---|---|
| Root Finding | Bisection, Newton-Raphson, Secant |
| Interpolation | Lagrange, Newton Forward, Newton Backward |
| Numerical Integration | Trapezoidal, Simpson 1/3, Simpson 3/8 |
| Differential Equations | Euler, Modified Euler |
| Regression | Linear, Exponential, Power, Quadratic |

## Features

- Dark customtkinter desktop UI
- Sidebar navigation
- Embedded matplotlib graphs
- Iteration/result tables
- Professional validation and error messages
- Modular OOP structure

## Requirements

- Python >= 3.10
- See `requirements.txt` for package dependencies
